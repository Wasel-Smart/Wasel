## Quick Start (Minimum Viable)

If you're in a hurry, the MINIMUM icons needed are:

1. **icon-192x192.png** - Android install
2. **icon-512x512.png** - Android splash
3. **apple-touch-icon.png** (180x180) - iOS install
4. **favicon.ico** - Browser tab

These 4 icons will make your app installable.
